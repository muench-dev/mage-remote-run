# Command Reference

mage-remote-run commands are grouped by resource type. Each command uses the active connection profile unless noted otherwise.

## Command Groups

- `connection`: Manage connection profiles and credentials.
- `website`: Manage websites.
- `store`: Manage store groups and store views.
- `customer`: Manage customers.
- `order`: Manage orders.
- `product`: Inspect product types and attributes.
- `eav`: Manage attribute sets.
- `tax`: Manage tax classes.
- `company`: Manage company information.

Run `mage-remote-run --help` to see all available commands and global options.
