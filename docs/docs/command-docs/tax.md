# tax

Manage tax classes.

## tax class list

List tax classes with paging.

```bash
mage-remote-run tax class list --page 1 --size 20
```

Options:

- `-p, --page <number>`: Page number
- `-s, --size <number>`: Page size

## tax class show `<id>`

Show tax class details.

```bash
mage-remote-run tax class show 2
```
