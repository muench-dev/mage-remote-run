# website

Manage Magento websites.

## website list

List all websites.

```bash
mage-remote-run website list
```

## website search `<query>`

Search websites by code or name (local filter).

```bash
mage-remote-run website search base
```

## website edit `<id>`

Edit website fields via interactive prompts.

```bash
mage-remote-run website edit 1
```

## website delete `<id>`

Delete a website after confirmation.

```bash
mage-remote-run website delete 2
```
