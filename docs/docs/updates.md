mage-remote-run follows npm release workflows. Update using your package manager or re-run via `npx` to get the latest published version.

## Update Global Install

```bash
npm update -g mage-remote-run
```

## Use the Latest with npx

```bash
npx mage-remote-run --help
```

## Verify Version

```bash
mage-remote-run --version
```
